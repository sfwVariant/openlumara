# Keyboard Shortcuts ⌨️

If you want to work like a pro, use these keyboard shortcuts to navigate and chat much faster!

## Essential Shortcuts
| Action | Windows/Linux | Mac |
| :--- | :--- | :--- |
| **Send Message** | `Enter` | `Enter` |
| **New Line (without sending)** | `Shift + Enter` | `Shift + Enter` |
| **Global Search** | `Ctrl + Space` | `Cmd + Space` |
| **Toggle Sidebar** | `Ctrl + B` | `Cmd + B` |
| **Clear Current Chat** | `Ctrl + L` | `Cmd + L` |
| **Open Settings** | `Ctrl + S` | `Cmd + S` |
| **Toggle Search Bar** | `Ctrl + F` | `Cmd + F` |

## Advanced Navigation
- **Escape Key**: This is your "get me out of here" button. It will close search bars, close pop-up menus, and even stop Lumara from generating a response if you want to cancel it.
- **Search Navigation**: Once you've performed a search, use `Enter` to go to the next result or `Shift + Enter` to go back to the previous one.
