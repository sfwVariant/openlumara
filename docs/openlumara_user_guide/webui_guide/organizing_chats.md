# Organizing Your Chats 📂

With many conversations happening, keeping things organized is key to a great experience.

## Searching for Conversations
Need to find that one specific chat from last week? 
- Use the **Search Bar** at the top of your chat list.
- You can search by the **Chat Title**.
- **Pro Tip**: If you enable "Search in Content," Lumara will even look through the actual messages to find exactly what you're looking for!

## Using Categories
Instead of one giant list, Lumara groups chats into categories.
- **General**: For your everyday chats.
- **Character Chats**: If you are talking to a specific persona, those chats are automatically grouped together.
- **Custom Categories**: You can move chats into different categories to keep your workspace tidy.

## Fast Loading
Don't worry about having hundreds of chats—the interface is built to be lightning-fast. Lumara only "loads" the chats that are actually visible on your screen, so the app stays smooth and responsive no matter how much history you build up.
