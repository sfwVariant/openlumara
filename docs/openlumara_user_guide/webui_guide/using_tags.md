# Using Tags to Stay Organized 🏷️

Tags are like little digital sticky notes that you can attach to your chats. They make finding specific topics incredibly easy.

## How to Use Tags
- **Filtering**: Once you've added tags to your chats, you can click a tag in your sidebar to see *only* the chats that have that label.
- **Quick View**: When you are searching, Lumara will show you which tags are associated with the results to help you identify them quickly.

## Why Use Them?
Tags are perfect for:
- **Project Work**: Label chats as `#work`, `#ideas`, or `#research`.
- **Moods**: Label chats as `#fun`, `#serious`, or `#learning`.
- **Characters**: If you have many different personas, use tags to group them.

## Pro Tip
You can also search for tags! If you type a tag name into the search bar, Lumara will find all chats containing that label.
